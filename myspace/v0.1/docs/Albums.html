GETalbums.getSupportedFields

Retrieves all supported fields.

Test console
Description
Parameter
string

Only returns the nearest multiple of 3 compared to the original value.

count

Optional

string

The following field names are supported. Bold are default values.
caption, id, location, mediaItemCount, mediaType, ownerId, thumbnailUrl
@all (a special value that tells the REST endpoint to send back all fields for the album.)

fields

Optional

string

MySpace specific field. May be one of the following values:
me - Album is private.
friendsonly - Album is visible to friends.
everyone - Album is visible to everyone.

msPrivacyLevel

Optional

string

Indicates the index of the first item to retrieve from the query set.

startIndex

Optional
