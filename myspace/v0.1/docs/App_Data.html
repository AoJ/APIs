GETappdata.getAllApplicationData

Retrieves all application data for a specified user.

Test console
Description
Parameter
string

The person's identifier.

personId

Required

string

Indicates which set of individuals to query for activities. @friends and @all currently can be used interchangeably.

selector

Required

string

The application's ID associated with the OAuth ConsumerKey/ConsumerSecret pair.

appId

Required

string

The fields correspond to keys in the application data. One uses fields to only retrieve specific keys. If the request omits fields, all application data per user will be returned.

fields

Optional
