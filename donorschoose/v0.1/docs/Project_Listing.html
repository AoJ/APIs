GETsearch

Project Search.

Test console
Description
Parameter
string

If you wish to wrap your response in a callback function, include the name here.

callback

Optional

string

By default, the start index of the result set is 0; for example, by setting index=10 and max=10, you can return rows 10-19.

index

Optional

string

By default, the maximum number of projects returned is 10; you can request up to 50 rows at a time.

max

Optional

string

To show the synopsis for each project listing set this to true.

showSynopsis

Optional

string

To show facet counts set this to true. They will appear after the project listings.

showFacetCounts

Optional

string

Facet counts will be displayed for all schools--including those with zero live projects--if this parameter is set to true and showFacetCounts is set to true and school counts are displayed because there is a community parameter in the search parameters.

showZeroFacetCountsForSchools

Optional

string

To search for a phrase, enclose it in quotes. It gets a list of projects where the keyword search matches any of the following fields:
- id
- title
- synopsis (full length project essay; not returned in JSON)
- short description
- fulfillment trailer
- school name
- teacher's first name (not returned in JSON)
- teacher's last name (only initial returned in JSON)
- city
- county (not returned in JSON)
- district (not returned in JSON)
- ward/neighborhood (not returned in JSON)
- state (e.g. TX)
- zip (not returned in JSON)
- subject name

You can sort your results by:

- Poverty: Shows projects from schools with the highest levels of poverty at the top - sortBy=1

- Popularity: Shows projects with higher numbers of donors at the top - sortBy=4
