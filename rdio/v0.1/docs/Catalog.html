POSTgetAlbumsByUPC

Finds and returns albums based on their Universal Product Code (UPC).

POSTgetAlbumsForArtist

Returns the albums by (or featuring) an artist.

POSTgetTracksByISRC

Finds and returns tracks based on their International Standard Recording Code (ISRC).

POSTgetTracksForArtist

Gets all of the tracks by this artist.

POSTsearch

Searches for artists, albums, tracks, users or all kinds of objects.

POSTsearchSuggestions

Matches the supplied prefix against artists, albums, tracks and people in the Rdio system. Return the first ten matches.
