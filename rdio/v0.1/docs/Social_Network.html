POSTaddFriend

Adds a friend to the current user.

POSTcurrentUser

Gets information about the currently logged in user.

POSTfindUser

Finds a user either by email address or by their username. Exactly one of email or vanityName must be supplied.

POSTremoveFriend

Removes a friend from the current user.

POSTuserFollowers

Gets a list of users following a user.

POSTuserFollowing

Gets a list of users that a user follows.
