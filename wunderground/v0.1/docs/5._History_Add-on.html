GETHistoryExample

This example will return historical data for San Francisco, California for the specified date. If you search by zip or lat/long you will get nearest airport.

GETHistoryExamplePws

This example will return historical data for a Personal Weather Station for the specified date. The feature only works with JSON. XML will come out soon.
