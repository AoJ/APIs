GETlistAccounts

Lists accounts.

Test console
Description
Parameter
string

The unique random token used for the graphical account creation process.

token

Required

string

Only return account associated to this email address.

email

Optional

string

Only return accounts with sources whose status is of a specific value. If an account has many sources, only those matching the given value will be included in the response. Possible statuses are: INVALID_CREDENTIALS, CONNECTION_IMPOSSIBLE, NO_ACCESS_TO_ALL_MAIL, OK, TEMP_DISABLED and DISABLED.

status

Optional

string

Set to 0 to get all accounts with sources that are not working correctly. Set to 1 for the opposite. As for the status filter above, only sources matching the specific value are included in the response.

status_ok

Optional

string

The maximum number of results to return.

limit

Optional

string
